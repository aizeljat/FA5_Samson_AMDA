# Group data by JobSatisfaction
groups = data.groupby('JobSatisfaction')

# Compute covariance matrices for each group
cov_matrices = [EmpiricalCovariance().fit(group[['Age', 'DailyRate', 'MonthlyIncome']]).covariance_ for _, group in groups]
n_samples = [len(group) for _, group in groups]

# Calculate pooled covariance matrix
pooled_cov = sum((n - 1) * cov for n, cov in zip(n_samples, cov_matrices)) / (sum(n_samples) - len(n_samples))

# Calculate Box's M statistic
M_statistic = (sum(n_samples) - len(n_samples)) * np.log(np.linalg.det(pooled_cov))
M_statistic -= sum((n - 1) * np.log(np.linalg.det(cov)) for n, cov in zip(n_samples, cov_matrices))

# Degrees of freedom and p-value for Box's M test
df = 0.5 * len(cov_matrices) * (len(cov_matrices) + 1) * (len(cov_matrices) - 1)
p_value = 1 - chi2.cdf(M_statistic, df)

print(f"Box's M statistic: {M_statistic}")
print(f"Degrees of freedom: {df}")
print(f"P-value: {p_value}")

Box's M statistic: 25.167946543107973
Degrees of freedom: 30.0

  • Box's M Statistic: The calculated Boxs M statistic is approximately 25.17. This statistic is used to test whether the covariance matrices of the dependent variables (Age, DailyRate, MonthlyIncome) are equal across groups defined by levels of JobSatisfaction.

  • Degrees of Freedom: The test has 30 degrees of freedom, which is determined based on the number of groups and variables in the dataset.

  • P-Value: The p-value is approximately 0.717. Since this value is greater than the conventional significance level (e.g., 0.05), we fail to reject the null hypothesis that the covariance matrices are equal across the groups.

Interpretation

A non-significant result (p > 0.05) suggests that there is no evidence to indicate a difference in covariance matrices across the levels of JobSatisfaction. This means that the assumption of homogeneity of covariance matrices is reasonably met. This finding is favorable for running further multivariate analyses, such as MANOVA, as it suggests that the groups have similar variance-covariance structures, which is an important assumption for many multivariate statistical tests.

    results = {}

from scipy.stats import levene

# Define a function to check homogeneity of variances using Levene's test
def check_homogeneity(df):
    homogeneity_results = {}
    for column in df.columns[:-1]:  # Exclude the grouping variable 'JobSatisfaction'
        # Perform Levene's test on each dependent variable across the levels of JobSatisfaction
        stat, p_value = levene(*[group[column] for name, group in df.groupby('JobSatisfaction')])
        homogeneity_results[column] = p_value
    return homogeneity_results

# Check homogeneity of variances for dependent variables
homogeneity_results = check_homogeneity(data[['Age', 'DailyRate', 'MonthlyIncome', 'JobSatisfaction']])

# Output the homogeneity test results (p-values)
